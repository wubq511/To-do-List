## 🚀 使用方法

1. **🌐 访问应用**：[https://responsive-to-do-web-tc4r.bolt.host/](https://responsive-to-do-web-tc4r.bolt.host/)
2. **📝 添加任务**：在输入框中输入您的待办事项
3. **✅ 确认添加**：点击添加按钮或按回车键确认
4. **🔄 状态切换**：完成任务后点击任务左侧的复选框标记为完成
5. **🗑️ 删除任务**：如需删除任务，点击任务右侧的删除按钮
